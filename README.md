## LCS Problem Hazards Detection & Visualization with RIPES
### Overview

This project uses RISC-V Assembly to visualize hazards detected in an implementation of a Longest Common Substring (LCS) alorithm with the open graphical processor simulator.

The uploaded png files shows how the pipelines got stalled in the simple in-order RISC-V processor. 

Install the Ripes simulator and use the provided LCS assembly code for more analysis.
